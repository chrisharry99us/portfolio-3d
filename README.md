# Chris Rodrigues — 3D Portfolio

An immersive 3D portfolio experience built entirely in Three.js.

## Live Site

[View Portfolio](https://portfolio-3d-flax.vercel.app)

## Tech

- Three.js r128
- Vanilla JavaScript
- HTML5 Canvas API
- CSS3

## Run locally

Download both files to the same folder and open `index.html` in Chrome or Edge.

> Note: Music (live jazz stream) only works when hosted online.
